# 📊 Sales Analysis Data Storytelling using Excel & Power BI

## 📌 Project Overview

This project was completed as **Task 4** of my **Data Analytics Internship at ApexPlanet**.

The objective was to present business insights from a sales dataset using effective data storytelling techniques supported by Microsoft Excel and Power BI.

---

## 🎯 Objectives

- Analyze sales performance
- Identify business trends
- Present insights professionally
- Improve decision-making through visualization

---

## 🛠️ Tools Used

- Microsoft Excel
- Power BI Desktop
- Microsoft PowerPoint

---

## 📂 Project Workflow

1. Data Collection
2. Data Cleaning
3. Data Validation
4. Exploratory Data Analysis (EDA)
5. Dashboard Creation
6. Business Insights
7. Data Storytelling Presentation

---

## 📈 Dashboard Highlights

- Sales by Category
- Monthly Sales Trend
- Top 10 Products by Sales
- Profit by Segment
- Sales by Region

---

## 💡 Business Insights

- Technology category generated the highest sales.
- Consumer segment contributed the highest profit.
- Top-selling products generated a major portion of total revenue.
- Sales showed seasonal trends across different months.
- Regional performance varied, creating opportunities for business growth.

---

## 📑 Presentation

The presentation explains the complete analytical workflow from raw data to business recommendations using clear visualizations and storytelling.
